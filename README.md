## Installation

1. Clone the repository

```sh
git clone https://github.com/GiovaniMenegat/tracking-api.git
```

2. Install the packages

```bash
bun install
#or
npm install
#or
pnpm install
#or
yarn install
```

4. Create the docker container

```bash
sudo docker compose up -d
```

3. Run the project

```bash
npm run start:dev
# or
yarn start:dev
# or
pnpm start:dev
# or
bun start:dev
```
